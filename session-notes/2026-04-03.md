2. **Standalone page** (`second-opinion.html`) — dedicated quiz page with its own SEO, structured data, and page header

### Exit Intent Popup Replaced
- Removed the old TidyCal calendar popup

### Navigation Updates
- **Nav CTA button** added across all 9 pages: secondary outline button ("Second Opinion / Assessment" on two lines) positioned left of "Book a Meeting"
- CSS handles all states: transparent navbar, scrolled navbar, non-transparent navbar, and mobile menu

### Form Submission — Dual Netlify Forms
- **`second-opinion`** — receives submissions when consent checkbox is checked (default)
- **`second-opinion-no-contact`** — receives submissions when consent is unchecked
- Both forms include: contact info (first/last name, email, phone, city, best time), all 8 quiz answers, gap count, result level, and consent status
- Hidden Netlify form declarations on `index.html` and `second-opinion.html` for build-time detection
- Submissions sent via background `fetch()` — results display immediately regardless
- **Post-deploy setup required:** Add email notifications in Netlify dashboard for both forms with different subject lines

### Bot Prevention (Quiz)
- **Honeypot fields:** Two hidden inputs (`website_url`, `email_confirm`) added to quiz contact form in all 3 locations
- **Time gate:** 15-second minimum before form submission is accepted (quiz realistically takes 1–3 minutes)
- **Interaction signal:** Requires at least one human interaction event (mouse, scroll, touch, keypress)
- **Graceful degradation:** Bots still see results but form data is silently blocked from Netlify

### TidyCal Booking Prefill — Removed
- Attempted prefilling TidyCal iframe with name/email via URL query params — did not work reliably
- Prefill code removed from both `booking-modal.js` and `second-opinion.js`
- Booking modal now opens cleanly on top of (and auto-closes) the exit popup quiz for a smooth transition

### Compliance & SEO Updates
- **Sitemap:** Added `second-opinion.html` (priority 0.8, lastmod 2026-04-03)

- **CLAUDE.md:** Updated page count and pages table
- **Structured data:** WebPage schema with BreadcrumbList on `second-opinion.html`
- **Meta tags:** Full OG, Twitter Card, description, canonical URL

### GA4 Quiz Analytics
- **Events tracked:** `sof_quiz_start`, `sof_question_answered` (per question with answer value), `sof_contact_form_viewed`, `sof_form_submitted` (with consent, gaps, result level, portfolio range, retirement timeline), `sof_results_viewed`, `sof_schedule_clicked`
- Every event tagged with `quiz_surface` (homepage_embed, standalone_page, or exit_popup)
- UTM parameters attached to all events when present
- Enables funnel analysis: start → each question → contact form → submit → results → schedule

### UTM Parameter Capture
- Reads `utm_source`, `utm_medium`, `utm_campaign`, `utm_term`, `utm_content` from URL query string
- Persists in `sessionStorage` so UTMs survive page navigation within the session
- Attached to both GA4 events and Netlify form submissions
- UTM fields added to all 4 hidden Netlify form declarations
- Example ad URL: `highridgeadvisory.com/second-opinion.html?utm_source=facebook&utm_medium=cpc&utm_campaign=spring-2026`
