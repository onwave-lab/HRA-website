## Notes

- During the audit I miscounted internal links to `second-opinion.html` (claimed 0, actual 10). Cause: my grep regex `href="[^"]*second-opinion\.html"` required the closing `"` immediately after `.html` and missed every link with a `#fragment`. Future SEO grep checks should match `href="[^"]*second-opinion\.html[^"]*"` to allow anchor fragments.
- "Discovered – currently not indexed" is a crawl-budget signal, not a linking signal. Adding more internal links to already-well-linked pages does not help. External backlinks accelerate it; manual GSC indexing requests nudge it.
- Google Indexing API is restricted to JobPosting/BroadcastEvent schemas — no programmatic way to request indexing for general pages. URL Inspection in GSC is the only path.
